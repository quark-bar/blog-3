Source code for my website! 

## Local setup

Create `env.sh` with your env variables, then run these commands:

```bash
npm install
npm develop
```

## Write a new post

```bash
npm install
./scripts/new-post.sh
```

## Deployment

Netlify deploys from the `master` branch
